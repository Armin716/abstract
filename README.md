## Instalación
Ejecutando un servidor local 
Asegúrese de tener Nodejs y NPM instalados
-npm -v
-node -v

Instalar Gatby CLI
-npm install gatsby-cli -g

Ejecute los siguientes comandos en el directorio principal
-npm install
-gatsby develop
